-----------Informations générales------------

1. L'application .jar se trouve sur le chemin  -> /Rendu3dProjet/out/artifacts/Rendu3dProjet_jar/

2. Pour l'executer utiliser la commande  " java -jar Rendu3dProjet.jar "

2.1 la méthode Main se trouve sur la classe Rendu3D

3. l'application demande la hauteur et la largeur de l'image -> le plan d'execution pour l'algorithme

4. l'application retourne l'image en question  avec le resultat final et le temps d'execution

5. vous avez un affichage (pop-up) de l'image en question.

6. le fichier test.txt doit respecter la meme façon d'écriture sinon une exception est levé

7. LE NON RESPECT DE LA FAçON D'ECRITURE provoque --> APPLICATION FAILS

8. Ne changez aucun emplacement sinon le parser ne  peux plus trouver son fichier.

9.exemple d'execution:

/*
commande: LinOES@LinuxTrashed:cd /Rendu3dProjet/out/artifacts/Rendu3dProjet_jar/
commande: java -jar Rendu3dProjet.jar

Donner largeur:800
Donner  hauteur:600
Au revoir,
In 5 seconds
*/
